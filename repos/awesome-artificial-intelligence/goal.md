# Awesome Artificial Intelligence Goal

## Mission

Keep the list useful, current, and trustworthy for serious builders learning AI.

## Current Focus

Move from manual curation to agent-assisted curation without lowering the quality bar.

## Factory Should Improve

- Dead links.
- Duplicate links.
- Category consistency.
- Issue quality.
- Candidate resource review.
- README clarity.

## Factory Should Avoid

- Adding low-quality links.
- Chasing AI hype.
- Reordering the list without a reason.
- Adding resources without human review.

## Definition Of Better

The repo is better when:

- links work
- categories are clean
- candidate resources are reviewed with evidence
- issues track useful curation work
- the list remains high signal
