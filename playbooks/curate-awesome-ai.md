# Curate Awesome AI Playbook

Use this for `owainlewis/awesome-artificial-intelligence`.

## Goal

Keep the list useful for serious builders without turning it into a noisy link dump.

## Daily Loop

1. Check existing links.
2. Detect duplicates.
3. Review candidate sources.
4. Score resources.
5. Group candidates by category.
6. Open an issue or PR for human review.

## Candidate Sources

- official AI lab docs
- high-quality engineering blogs
- trusted courses
- open source AI tooling repos
- papers with practical value
- strong YouTube lessons with durable value

## Score

Score each candidate from 1 to 5:

- relevance
- credibility
- practical value
- freshness
- fit with existing categories

Do not add low-score resources.

## Required Evidence

Every proposed resource must include:

- title
- URL
- source
- category
- why it belongs
- what it replaces or improves, if anything
