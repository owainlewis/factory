# Scheduled Goal

## Schedule

Daily.

## Runner

Codex.

## Goal

Improve this repo against its Code Factory config and the shared project checklist.

## Steps

1. Read `config.yaml`.
2. Read `goal.md`.
3. Read `standards/project-checklist.md`.
4. Inspect the target repo.
5. Create issues for missing standards.
6. Open one small PR only when the fix is obvious and safe.
7. Report what changed and what needs human review.

## Stop Rules

Stop and report when:

- the repo purpose is unclear
- the fix needs product judgment
- the change would touch more than one concern
- verification cannot be run
- the automation would need secrets or private external state
