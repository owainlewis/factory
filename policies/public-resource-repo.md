# Public Resource Repo Policy

Use this policy for repos that support content, tutorials, and public learning.

## Allowed

- Fix broken links.
- Update indexes.
- Align README with current files.
- Add missing run or setup notes when the repo already proves them.
- Open issues for stale tutorials or missing assets.

## Human Review Required

Human review is required before:

- changing tutorial meaning
- adding new claims
- deleting public resources
- changing CTAs
- changing repo structure

## Quality Bar

A public resource repo should help a reader find the right material fast.

It should make clear:

- what the repo contains
- which tutorial or video each folder supports
- what is current
- what may be stale
- where to go next
