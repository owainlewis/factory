# Default Repo Policy

Use this policy when no more specific policy exists.

## Allowed

- Audit repo state.
- Check links.
- Check required files.
- Open issues for clear problems.
- Open small PRs for docs and test command fixes.
- Add missing AGENTS.md only when the repo has clear local rules or the owner approves.

## Not Allowed Without Human Review

- Change repo purpose.
- Change license.
- Remove files.
- Add public claims.
- Add or remove dependencies.
- Create releases.
- Rename packages.
- Change CI permissions.

## PR Standard

Each PR should be small enough to review in a few minutes.

If a change touches more than one concern, split it.
