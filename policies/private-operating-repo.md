# Private Operating Repo Policy

Use this policy for private repos that coordinate strategy, workflows, and operating context.

## Allowed

- Detect stale references.
- Align docs when the source of truth is clear.
- Open issues for contradictions.
- Improve existing docs when requested.

## Human Review Required

Human review is required before:

- changing strategy
- creating new docs
- changing workflows
- deleting context
- moving files
- changing source-of-truth files

## Quality Bar

Private operating repos should reduce surface area.

They should make the next action obvious and avoid becoming a junk drawer.
